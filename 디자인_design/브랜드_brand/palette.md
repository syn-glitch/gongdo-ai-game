## 3. 카툰 스타일 규정

- **아웃라인**: 모든 주요 UI 요소에 **카툰 블랙 `#1A1A1A` 2~3px** 테두리 적용
- **라운드**: 버튼·카드 `border-radius: 12px~16px` (초등 친화 부드러움)
- **그림자**: `box-shadow: 4px 4px 0 #1A1A1A` (카툰 오프셋 섀도)
- **호버**: 버튼 hover 시 `transform: translate(-2px, -2px)` + 그림자 `6px 6px`
