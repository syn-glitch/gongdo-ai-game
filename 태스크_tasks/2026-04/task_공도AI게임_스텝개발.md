## 1. 학습 패러다임 (v2 확정)
**하네스(Harness) 프롬프트 문서 방식**
1. 학생이 좌측 드로어에서 **차시 폴더** 선택 → 해당 **바이브코딩 문서** 로딩
2. 문서를 읽고 (필요시 수정) → **[시작 ▶]** 클릭
3. Claude API가 문서를 해석 → HTML 게임 생성 → 우측에서 즉시 실행
4. 문서 수정 → 재시작 → 변경 확인 **반복**
5. 막히면 하단 **AI튜터(공도쌤)** 에게 질문
