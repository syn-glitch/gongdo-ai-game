## 0. 지시 원문

## 1. 왜 이 작업이 필요한가
현재 공도 AI-Game 앱은 **이모지**(🦸 👒 🐰 🦁)로 캐릭터를 표시하고 있습니다 (S10~S12 임시 폴백). 학생 경험은 완결되어 있으나, 넷마블 IP 교육 프로그램의 **브랜드 완성도** 를 위해 실제 넷마블 프렌즈 캐릭터 이미지가 필요합니다.

### 자동 전환 메커니즘
앱 코드에는 이미 **onerror 폴백** 로직이 들어가 있어, PNG 파일이 지정 경로에 배치되면 **앱 재시작·코드 수정 없이** 이모지 → 실제 이미지로 자동 전환됩니다. 브라우저 새로고침만 하면 됩니다.

## 4. 파일 규격

| 속성 | 값 |
|:---|:---|
| 포맷 | **PNG** (JPEG 불가) |
| 크기 | **정확히 512 × 512 px** |
| 배경 | **투명** (알파 채널 포함) |
| 색 공간 | sRGB |
| 캐릭터 배치 | **중앙 정렬**, 상하좌우 여백 ≈ 각 32px (보호 여백 규정) |
| 아웃라인 | 원본 카툰 스타일 그대로 (2~3px 검정) |
| 필터·효과 | **금지** (색 보정·그림자·블러 등 일체 적용하지 않음) |

### 금지 사항 (IP 매뉴얼 준수)
- ❌ 원본 색상 변경
- ❌ 비율 왜곡 (가로세로 늘이기·줄이기)
- ❌ 얼굴·표정 편집
- ❌ 의상·소품 변형
- ❌ 밝기·채도 조정

## 7. 작업 흐름 (권장 순서)

1. Adobe Illustrator 에서 원본 `.ai` 열기
2. 캐릭터별 레이어 격리
3. 512×512 아트보드 생성
4. 각 캐릭터를 아트보드 중앙 정렬 + 여백 32px 확보
5. **파일 → 내보내기 → PNG** (alpha 체크, 해상도 72dpi)
6. 파일명 정확히 입력
7. 지정 폴더에 4개 모두 배치
8. 브라우저 새로고침으로 앱 동작 확인
9. 본 문서 하단에 **완료 보고** 추가
