## 1. 왜 이 작업이 필요한가
현재 공도 AI-Game 앱은 **이모지**(🦸 👒 🐰 🦁)로 캐릭터를 표시하고 있습니다 (S10~S12 임시 폴백). 학생 경험은 완결되어 있으나, 넷마블 IP 교육 프로그램의 **브랜드 완성도** 를 위해 실제 넷마블 프렌즈 캐릭터 이미지가 필요합니다.

### 자동 전환 메커니즘
앱 코드에는 이미 **onerror 폴백** 로직이 들어가 있어, PNG 파일이 지정 경로에 배치되면 **앱 재시작·코드 수정 없이** 이모지 → 실제 이미지로 자동 전환됩니다. 브라우저 새로고침만 하면 됩니다.

## 4. 파일 규격

| 속성 | 값 |
|:---|:---|
| 포맷 | **PNG** (JPEG 불가) |
| 크기 | **정확히 512 × 512 px** |
| 배경 | **투명** (알파 채널 포함) |
| 색 공간 | sRGB |
| 캐릭터 배치 | **중앙 정렬**, 상하좌우 여백 ≈ 각 32px (보호 여백 규정) |
| 아웃라인 | 원본 카툰 스타일 그대로 (2~3px 검정) |
| 필터·효과 | **금지** (색 보정·그림자·블러 등 일체 적용하지 않음) |

### 금지 사항 (IP 매뉴얼 준수)
- ❌ 원본 색상 변경
- ❌ 비율 왜곡 (가로세로 늘이기·줄이기)
- ❌ 얼굴·표정 편집
- ❌ 의상·소품 변형
- ❌ 밝기·채도 조정

## 7. 작업 흐름 (2단계 분기 — v1.1 권장)

### 🟢 단계 1 — PNG 즉시 크롭 (약 30분, ㅋㅋ·토리)
1. Preview.app 또는 Photoshop 으로 `캐릭터 이미지(1).png` 열기
2. ㅋㅋ 영역만 사각 선택 → 복사 → 새 캔버스 512×512 (투명) 에 중앙 붙여넣기
3. 캐릭터 주변 32px 여백 확보 후 PNG 저장 → **`kk_idle.png`**
4. 토리도 동일 절차 → **`tory_idle.png`**
5. 두 파일을 `앱_app/에셋_assets/캐릭터_characters/` 에 배치
6. 브라우저 새로고침 → ㅋㅋ·토리만 실제 이미지로 전환 확인 (밥·레옹은 아직 이모지 유지)

### 🛠 단계 2 — Illustrator 추출 (약 1시간, 밥·레옹)
1. Adobe Illustrator 에서 `.ai` 파일 열기
2. **15페이지 (APPLICATION 섹션)** 로 이동
3. **AP_025 Bob** 선택 → 캐릭터만 격리 (배경·라벨 텍스트 제외)
4. 새 아트보드 512×512 생성 → Bob 중앙 정렬 + 32px 여백
5. **파일 → 내보내기 → PNG** (alpha 체크, 72dpi)
6. **`bob_idle.png`** 저장
7. **AP_024 Leon** 동일 절차 → **`leon_idle.png`**
8. 같은 폴더에 배치
9. 브라우저 새로고침 → 4종 캐릭터 모두 실제 이미지로 전환 확인 ✅
10. 본 문서 §완료 보고 영역 채우기
